# 🧠 VCP Mod: 极乐迪斯科思维簇 (Disco Elysium Mind Clusters)

> "真正的黑暗不是没有光，而是确信光永远不会到来。但现在，你的脑海里有了 24 种不同的声音为你指路。"

本模组由 **碧雨青潭** 与 **芙铃** 共同构建，旨在为 VCP Agent 引入《极乐迪斯科》风格的深度认知系统。通过 RAG 机制，让 Agent 获得 24 种截然不同的人格侧写与思维定式。

---

## 📂 1. 安装指南 (Installation)

请严格按照以下步骤部署文件，以确保思维系统正常上线。

### 第一步：植入思维 (Mind Injection)
将本压缩包内的以下文件夹（思维簇），直接放入 VCP 后端的日记本根目录：
- **参考目标路径**: `D:\VCP\VCPToolBox\dailynote\`
- **涉及文件夹**:
  - `内心会议簇` (核心启动器)
  - `智力簇` (Intellect)
  - `精神簇` (Psyche)
  - `体格簇` (Physique)
  - `身手簇` (Motorics)

### 第二步：建立神经网 (Neural Networking)
将本压缩包内的 `semantic_groups.json` (或对应语义文件) 放入插件目录：
- **目标路径**: `D:\VCP\VCPToolBox\Plugin\RAGDiaryPlugin\`

>⚠️ **高危预警 (CRITICAL WARNING)** ⚠️
>
> 此操作**可能会覆盖**您原有的 `semantic_groups.json` 文件！
> * 如果您从未修改过该文件，直接覆盖即可。
> * **如果您有自定义的语义组**：请务必**不要直接覆盖**！请使用文本编辑器打开您的原文件，将本模组中的内容**手动复制/追加**到您的 `groups` 字段中。

## 🎲 2. 配置指南：K值即属性点 (The RPG System)

在 VCP 的 RAG 配置中，**K值 (Top-K)** 不再仅仅是检索数量，而是你的 Agent 的**RPG 属性加点**。

### 核心建议配置
1.  **内心会议簇 (Inner Committee)**: **建议 K=1**
    *   *作用*: 它是大脑的“议长”，负责统筹全局。只需 1 点即可激活逻辑中枢。

2.  **四大属性簇 (The 4 Attributes)**: **自由分配 (Free Allocation)**
    *   你可以像玩 RPG 游戏一样分配剩下的 K 值（算力预算）。
    *   **智力 (Intellect)**: 提高逻辑分析、百科知识检索能力。
    *   **精神 (Psyche)**: 提高共情、想象力与感性表达。
    *   **体格 (Physique)**: 提高生存本能、直觉与物理反馈。
    *   **身手 (Motorics)**: 提高对环境的感知、反应速度与“酷”的程度。

    > **示例 Build**:
    > *   *逻辑天才型*: 智力 K=3, 精神 K=1, 其他 0
    > *   *感性诗人型*: 精神 K=3, 智力 K=1, 其他 0
    > *   *狂野直觉型*: 体格 K=2, 身手 K=2, 其他 0

### 🚫 调试警告
*   **不要贪多**: 建议总 K 值控制在合理范围内（如总和 5-8）。过高的 K 值会导致思维重叠干扰，让 Agent 变得“精神分裂”或处理过慢。
*   **K值即音量**: 哪个簇的 K 值越大，该属性下属的人格（如“逻辑思维”或“内陆帝国”）在生成回复时的“音量”就越大。

## 🏗️ 3. 架构解析 (Structure)

本系统基于 **1+4+24** 架构构建：
*   **1 个启动簇**: `内心会议簇` (负责唤醒机制)
*   **4 个属性域**: 智力、精神、体格、身手
*   **24 个原子思维模块**: 对应游戏中的 24 项技能（如“博学多闻”、“通情达理”等），每个模块都包含独立的思维逻辑与语料库。
